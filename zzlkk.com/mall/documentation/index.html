Introduction

We offer an excellent solution for those who want to improve their rental and real estate business and manage it in order to obtain better results. Due to its flexible and modular architecture, the script can be used for different businesses based on nightly, weekly monthly rent.

Features

  • Three types of rental system Nightly / Weekly / Monthly in one script.
  • Real time availability and booking.
  • Assign unlmited apartment/property.
  • Assign unlimited property/partment  types.
  • Unlimited image upload of a apartment/property.
  • Upload floorplan of apartment/property.
  • Assign unlimited apartemnt/property facilities/amenities
  • Assign each aprtement/property price as regular and specail price upon duration.
  • Set Advance amount percent based on duration of rent.
  • Apartment Blocking in particular date&tme range. i.e. booking from desk and phone blocking feature will be use.
  • Booking list of each apartment  by active booking, booking history, by customer.
  • Availability yearly calendar view of each apartment/property.
  • Content management system of frontend where you can create new menu, page and assign page content by advance ckeditor.
  • PayPal and Manual gateway integrated.
  • Printable Invoice from booking list.
  • After successful booking email  notification both site owner and customer  with invoice.
  • Confirmation and cancellation email content can be edit from admin panel.
  • Any currency support, different date format,  site time zone setting,   set taxes.
  • You can set booking type nightly or weekly or yearly.
  • can be set minimum and maximum stay of each booking type.
  • weekly booking can be set start day of week.
  • front-end used google map v3 for apartment map.
  • Advance availability calendar for each apartment integrated in fron-end.
  • All apartment search by type/city/zipcode
  •  Measurement unit can be edit
  • View map on search result
  • Listing of today booking, today check-in, today check-out in admin home page
  • SEO friendly url for view apartment details (htaccess)

Installation

Step1:   Unzip ' apartment-rental.zip'  file.

Step 2:  Upload apartment-rental '  folder in your server. also you can make your own folder name.

Step3:  Enter URL of uploaded folder . example- http://www.yoursite.com/apartment-rental/

Step4:  Installation procedure page will display. just enter database details  and follow the wizard.

Step5:  Removed 'Install' folder after successfully install script

Your  admin panel:  http://www.yoursite.com/apartment-rental/cp/

 

Troubleshot:

If page not found in view apartement details:

htaccess write based root directory of your server. maximum server work without change htaccess.

if you install script in any sub folder , you may get apartment details "page not found" error.

so just edit htaccess file in script directory. for example you installed script "apartment-rental" folder in your site.

so edit htaccess file line number by Notepad: 4

RewriteBase /apartment-rental/

Setup Rental Type(Important and First step)

First Login in your Admin Panel. You can see following Option under "Setting > Apartment Booking Type"

Before entry apartment inventory select you Rental Type Nightly or Weekly or Monthly.

You need to select anyone among them from combo box.

Here you can define maximum stay and minimmum stay. 

For weekly rental you can set start day of week.

 

Important: If you change rental type after set up all apartment enventory then price plan will be reset for all apartment. example: you have currently nightly rental, in the middle you changed rental type to Weekly , so all price plan for nightly rental will be reset to 0.00, you need to edit them for weekly price.

How can manage apartment/property?

First Login in your Admin Panel. You can see following Option under "Apartment Manager"

  • Apartment Types: First insert all type of partment of have. like town house, open house, muti storied building.
  • Apartment Facilities: Second insert all apartment facilities/amenities.
  • Apartment List: Third click on 'Add New Apartment' button. here you can entry apartement with all facilities and features. also you can uplaod master image and floorplace from here.
  • Apartment Gallery: Last select apartment from combo box and upload your images like room, bathroom etc.

How can manage price?

First Login in your Admin Panel. You can see following Option under "Price Manager > Price Plan"

Select partment from combo box.

At the time of apartment entry default price plan insert with 0.00 value. you need to edit default price by click edit link.

you inser new price based on stay duration of each apartment.

How can manage Booking?

First Login in your Admin Panel. You can see following Option under "Booking Manager"

  • View Booking List:  Booking list devided into active booking & booking history. you need select  apartment and type of booking need to list from dropdown combo. from active booking list you can print invoice, cancel booking, view booking details. From booking history you can booking booking details and delete booking forever.
  • Customer Lookup:  Here you can list of all customer details and can edit details. also you can view booking list of seelcted customer.      
  • Calendar View:  Here you can see each apartment yearly calendar view of availability .
  • Apartment Blocking:   Apartment Blocking in particular date range. i.e. booking from desk and phone blocking feature will be use.
